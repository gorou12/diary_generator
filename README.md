# DIARY GENERATOR

## HOW TO DEVELOPMENT

try this

```bash
# install uv
curl -LsSf https://astral.sh/uv/install.sh | sh
exec $SHELL -l

# cd this repository and
uv sync
```

Run local debug server: `uv run python -m http.server 8000 --directory output`
