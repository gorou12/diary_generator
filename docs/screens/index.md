# トップページ

## メインコンテンツ部構成

* h2見出し「最新の日記」
* 大ブロック
    * h3見出し　年月日
    * 小ブロック
        * h4見出し トピック名
        * タグ
        * 本文

## コンテンツ部詳細

* h3見出し年月日
    * 年月日はyyyy-mm-ddで表示
    * その右横に📅アイコン（[日付ページ](/docs/screens/dates/yyyy-mm-dd.md)へのリンク）
* h4見出しトピック名
    * その右横に🔍アイコン（[トピックページ](/docs/screens/topics/topic-name.md)へのリンク）
* タグ
    * 横一列に並ぶ
    * タグごとにspanで囲む
    * タグ名の前にタグアイコンをつける
    * span全体を[トピックページ](/docs/screens/topics/topic-name.md)へのリンクをつける
