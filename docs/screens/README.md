# Diary Generator 画面設計

## 画面一覧

| 画面名 | URL例 | 目的 |
|---|---|---|
| [トップページ](/docs/screens/index.md) | `/` | 最新のトピックを表示する |
| [日付一覧ページ](/docs/screens/dates/index.md) | `/dates/` | 日付ページを並べた一覧ページ |
| [日付ページ](/docs/screens/dates/yyyy-mm-dd.md) | `/dates/{yyyy}-{mm}-{dd}/` | 当日の記事を表示するページ |
| [トピック一覧ページ](/docs/screens/topics/index.md) | `/topics/` | トピック名を並べた一覧ページ |
| [トピックページ](/docs/screens/topics/topic-name.md) | `/topics/{topic-name}/` | 特定トピックの記事を表示するページ |
| [検索ページ](/docs/screens/search.md) | `/search/` | キーワードで記事を検索するページ |

## 共通部品一覧

| パーツ名 | 目的 |
