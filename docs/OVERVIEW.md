# Diary Generator 機能仕様書

## 概要

このプログラムは、静的生成のブログ風サイトを出力するプログラム。

## 言葉の定義

- トピック
    - タイトル、タグ、本文、その他必要なメタデータを含む1つの塊のことを指す。
    - 記事とも呼ぶ。
    - タイトルのことを「トピック名」とも呼ぶ。
- タグ
    - トピックに含むことがある0個以上の識別子。
    - 入力時は#から始まるため、「ハッシュタグ」とも呼ぶ。


## 主な仕様

- トピック名やタグで絞り込むことができる
    - 具体的にはトピック名やタグごとに個別ページ「トピックページ」を設け、当該トピック名・タグの記事のみを集めたページを用意する。
    - このプログラムにおいて「トピック名」と「タグ」に差は設けず、どちらもあたかも同じものとして扱う。
        - すなわち、「天気」という名のトピックも、「雨の日」という名でタグ「#天気」を含むトピックも、いずれも「天気」のトピックページに等しく表示する。
        - これは記入時に何を題名にするか？で迷わないようにするため。
- トピック名あるいはタグが「非公開」であるトピックは収集・公開対象としない
    - 個人情報を含む恐れのある日記のため。
    - 取り込み段階で除き、キャッシュデータにも残さない。
        - 記入媒体であるNotion側で見ることを想定している。
